## 🏁 Getting Started <a name = "getting_started"></a>

Para crear un ptoyecto angular:

```
ng new miApp
```

Para iniciar el servidor de desarrollo:

```
ng serve
```

Si se clono el repositorio.

```
npm install
```

### Prerrequisitos

Instalar node js.

Instalar angular cli:

```
npm install -g @angular/cli
```
